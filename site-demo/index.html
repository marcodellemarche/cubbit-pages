How it works

StepWhat happens
1. UploadCLI encrypts every file with AES-256-GCM and uploads .enc blobs to Cubbit S3
2. LoginBrowser downloads the public index.html (login page) and verifies your password against _verify.enc
3. Service WorkerSW intercepts all fetch requests and decrypts .enc files on the fly using the Web Crypto API
4. NavigateLinks work normally — the SW transparently decrypts every asset before the browser sees it
