About this demo

You navigated from Dashboard to About without re-entering the password. The Service Worker intercepted the request and decrypted about.html.enc and style.css.enc transparently.

Tech stack

ComponentTechnologyNotes
StorageCubbit S3EU-based, distributed, S3-compatible
EncryptionAES-256-GCMAuthenticated encryption, client-side only
Key derivationPBKDF2-SHA256100 000 iterations, random salt per file
DecryptionWeb Crypto APINative browser API, no JS library needed
InterceptionService WorkerTransparent; password cached in IndexedDB
CLIGo binarySingle binary, zero runtime deps, cross-platform

Deploy this demo yourself

CommandDescription
cubbit-pages setup Save credentials once
cubbit-pages deploy ./site --encrypt --password "your-password" Encrypt and deploy
cubbit-pages open --bucket my-bucket Open the deployed site in the browser
