Features

Deploy in un comando

Punta a una cartella, specifica il bucket. Cubbit Pages carica tutto con i MIME type corretti.

Cifratura AES-256-GCM

Ogni file cifrato lato client. Pagina di login generata automaticamente. Decifratura nel browser via Web Crypto API.

Upload paralleli

Upload concorrenti configurabili. ACL per-file o bucket policy. Dry-run per verificare prima.

Cross-platform

Binario Go standalone. Linux, macOS, Windows. AMD64 e ARM64. Zero dipendenze runtime.

Snippet pronti

Genera automaticamente bucket policy, CORS, IAM policy e lifecycle con un comando.

Open Source

Codice su GitHub. CI/CD con GitHub Actions. Progetto companion di Cubbit Seal.

Installazione

Script automatico

curl -sSL https://s3.cubbit.eu/cubbit-pages-releases/install.sh | bash

Build da sorgente

git clone https://github.com/marcodellemarche/cubbit-pages.git
cd cubbit-pages
make build

Quick start

1

Configura il bucket

cubbit-pages snippets --bucket mio-bucket
2

Deploy in chiaro

cubbit-pages deploy ./mio-sito \
  --bucket mio-bucket \
  --access-key $CUBBIT_ACCESS_KEY \
  --secret-key $CUBBIT_SECRET_KEY \
  --public-bucket
3

Oppure deploy cifrato

cubbit-pages deploy ./mio-sito \
  --bucket mio-bucket \
  --access-key $CUBBIT_ACCESS_KEY \
  --secret-key $CUBBIT_SECRET_KEY \
  --encrypt --password "la-mia-password"
